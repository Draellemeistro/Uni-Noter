# Wi-Fi Protected Access (WPA)
> [!NOTE] TLDR
> - 256-bit key
> - unlike **[[WEP]]**, clients don't use the same key
> - **RC4** is still used, but now calls it **TKIP**
> - **TWO MODES:**
> 	- Pre-shared Key (**PSK**): *Password*
> 	- Enterprise Mode: Authenticate with a *username* and *password*
