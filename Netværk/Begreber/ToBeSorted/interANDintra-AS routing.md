# Intra-AS routing
### [[RIP]]: 
- Classic DV: DVs exchanged every 30 secs
- no longer widely use
### [[EIGRP]]: 
- DV based
- Formerly Cisco-proprietary for decades (open in 2013)
### [[OSPF]]: 
- Link-state routing
- IS-IS protocol (ISO std, not RFC std) essentially same as OSPF
