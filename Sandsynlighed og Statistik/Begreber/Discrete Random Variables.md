
> [!NOTE] Definition
> a r.v. whose set of possible values is a sequence is said to be **discrete**
> For a discrete r.v. we define [[Probability Mass Function (PMF)]]
> - $p(a)=P(X=a)$
> - $p(x_{i})>0,i=1,2,\dots$
> 	- $p(x)=0, otherwise$
> - $\sum^{\infty}_{i=1}p(x_{i})=1$
